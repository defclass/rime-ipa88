# RIME IPA88 Schema

A rime schema to type IPA 88 .



## INSTALL

* Copy the two files into Rime configuration direcotry .

```
# eg: 
Macos: ~/Library/Rime
```

* add `schema: ipa88` into `schema_list` section.

```
patch:
  schema_list:
    - schema: ipa88
```

* click `Deploy` on Rime menu and wait for ready.
